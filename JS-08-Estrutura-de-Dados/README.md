# JS-08 e Java-8 - Estrutura de Dados

Soluções em JavaScript para as atividades de Estrutura de Dados (Fila e Pilha).

## Atividade 01 - Fila (`fila.js`)

Implementação de uma Fila (Queue) para simular a organização de clientes de um banco por ordem de chegada.

### Funcionalidades:
1. Adicionar um novo Cliente na fila.
2. Listar todos os Clientes na fila.
3. Chamar (retirar) uma pessoa da fila.
4. Finalizar o programa.

## Atividade 02 - Pilha (`pilha.js`)

Implementação de uma Pilha (Stack) para organizar a retirada de livros.

### Funcionalidades:
1. Adicionar um novo livro na pilha.
2. Listar todos os livros da Pilha.
3. Retirar um livro da pilha.
4. Finalizar o programa.

## Como Executar

Para executar os arquivos, certifique-se de ter o Node.js instalado e execute o comando no terminal:

```bash
node fila.js
# ou
node pilha.js
```

**Nota:** O módulo `readline-sync` é necessário para a interação via terminal.
```bash
npm install readline-sync
```
